<h2>What is PromptSync?</h2>
<p>
  PromptSync helps you evaluate and compare multiple LLMs at once. Send a single prompt and review responses from each configured model in a structured, side-by-side layout. It’s designed for quick iteration, model benchmarking, and sharing insights across your team.
</p>

<h2>Key Features</h2>
<ul>
  <li><strong>Side-by-side comparisons</strong>: View multiple model outputs in a clean grid.</li>
  <li><strong>Multi-model support</strong>: Add OpenRouter-compatible LLMs with model IDs and API keys.</li>
  <li><strong>Status at a glance</strong>: Each response includes per-model status and timing indicators.</li>
  <li><strong>Copy-friendly</strong>: One-click copy to share or archive responses.</li>
  <li><strong>Auth built-in</strong>: Email/password authentication to keep your workspace private.</li>
</ul>

<h2>Why PromptSync?</h2>
<ul>
  <li><strong>Faster evaluation</strong>: Test multiple providers and models in a single pass.</li>
  <li><strong>Consistent UI</strong>: Elegant, responsive design that scales from mobile to desktop.</li>
  <li><strong>Straightforward</strong>: Focus on writing prompts and reading results—no complex workflows.</li>
</ul>
